## Fastest Way to Get Online (5 minutes!)

### Using GitHub Pages (100% Free, No Credit Card)

1. **Create GitHub Account** (if needed)
   - Go to https://github.com
   - Sign up for free

2. **Create New Repository**
   - Click the "+" in top right â†’ "New repository"
   - Name it: `hr-parlay` (or anything you want)
   - Make it Public
   - Click "Create repository"

3. **Upload Your Files**
   - Click "uploading an existing file"
   - Drag ALL the files from this folder
   - Click "Commit changes"

4. **Enable GitHub Pages**
   - Go to Settings (top of repo)
   - Scroll to "Pages" in left sidebar
   - Under "Source", select "main" branch
   - Click Save
   - Wait 2-3 minutes

5. **Get Your URL**
   - Your app will be live at:
   - `https://YOUR-USERNAME.github.io/hr-parlay/`

## Update Park Factors Seasonally

Edit `parkFactors.js` and update the numbers from:
- Baseball Savant: https://baseballsavant.mlb.com/leaderboard/statcast-park-factors
- Just change the `factor` values (100 = average)
